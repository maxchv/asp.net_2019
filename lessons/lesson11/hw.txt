Написать веб приложение для вычисления факториала числа.
В контроллере HomeController добавить метод-действие Factorial, 
который принимает целочисленный аргумент для которого необходимо вычислить факториал.

Для оптимизации работы приложения создать кеширующий фильтр, который позволит
минимизировать время расчета факториала.
Данные кеша хранить в памяти.

Создать дополнительное представление Info в котором отобразить статистику
использования кеша: сколько было совершено запросов, сколько было вычилсений,
сколько раз использовались данные из кеша

Создать приложение для добавления, редактирования и отображения товаров.
Должна присутствовать область Admin, в которую добавить контроллеры: Home, Product и Category

В контроллеры Product и Category добваить действия: 
	Add (добавление товара/категрии), Edit - редактирование
В контроллере Home области Admin добавить две ссылки - на добавление товара и категории.

На главной странице (контроллер /Home, действие Index) отобразить все добавленные товары
В ответ на действие Details про параметру id - выдается детальная информация о товаре по id

По дополнительному ключу ?Category=<category_name> при запросе на контроллер Product (вне области)
отображается список товаров только заданной категории category_name.
